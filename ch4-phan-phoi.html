📦 Cách Phân phối Skill

Cá nhân

  1. Nén thư mục skill thành file .zip
  2. Vào Claude.ai → Settings → Capabilities → Skills
  3. Tải file ZIP lên → Claude tự nhận diện và kích hoạt

Tổ chức (Enterprise)

  • Quản trị viên triển khai skill toàn workspace
  • Cập nhật tự động, quản lý tập trung
  • API endpoint /v1/skills cho tích hợp lập trình
🌐
Skill là tiêu chuẩn mở — hoạt động trên Claude.ai, Claude Code và API. Tạo 1 lần, dùng mọi nơi.

🔧 Xử lý Sự cố Thường gặp

❌
"Could not find SKILL.md"
Đổi tên file thành đúng SKILL.md (phân biệt hoa thường).
❌
"Invalid frontmatter"
Kiểm tra dấu phân cách --- và dấu ngoặc kép trong YAML.
❌
"Invalid skill name"
Tên có khoảng trắng hoặc viết hoa. Dùng kebab-case.
💡
Skill không kích hoạt? Sửa lại trường description — thêm cụm từ mà người dùng thực tế sẽ nói.

📋 Checklist Hoàn chỉnh

📐 Trước khi bắt đầu

  • ☐ Xác định 2-3 use case
  • ☐ Xác định công cụ cần thiết
  • ☐ Xem ví dụ skill có sẵn
  • ☐ Lên kế hoạch cấu trúc thư mục

🔨 Trong quá trình phát triển

  • ☐ Thư mục đặt tên kebab-case
  • ☐ File SKILL.md đúng chính tả
  • ☐ YAML có dấu phân cách ---
  • ☐ Description có CÁI GÌ + KHI NÀO

🧪 Trước khi tải lên

  • ☐ Test kích hoạt — tác vụ rõ ràng
  • ☐ Test kích hoạt — yêu cầu diễn đạt lại
  • ☐ Không kích hoạt chủ đề không liên quan
  • ☐ Test chức năng đạt

🚀 Sau khi tải lên

  • ☐ Test trong hội thoại thực
  • ☐ Giám sát kích hoạt thiếu/thừa
  • ☐ Thu thập phản hồi
  • ☐ Cải tiến & cập nhật version
