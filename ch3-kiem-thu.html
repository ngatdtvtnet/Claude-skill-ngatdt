✅ Test 1: Kiểm thử Kích hoạt

Mục tiêu: Skill được tải đúng lúc, không tải khi không cần.

✅ NÊN kích hoạt

  • Tác vụ rõ ràng: "Tạo báo cáo tuần"
  • Diễn đạt lại: "Làm weekly report đi"
  • Ngữ cảnh gián tiếp: "Sếp cần bản tóm tắt"

❌ KHÔNG NÊN kích hoạt

  • Chủ đề không liên quan
  • Tác vụ thuộc skill khác
  • Câu hỏi chung về AI

⚙️ Test 2: Kiểm thử Chức năng

Mục tiêu: Skill tạo ra đầu ra đúng và xử lý được lỗi.

  1. Đầu ra hợp lệ: File được tạo đúng format, đúng nội dung
  2. API thành công: Các lệnh gọi MCP không thất bại
  3. Xử lý lỗi: Khi thiếu dữ liệu, skill hỏi lại thay vì crash
  4. Trường hợp biên: File rỗng, dữ liệu thiếu, format lạ

Mục tiêu: Chứng minh skill cải thiện kết quả so với không dùng skill.
